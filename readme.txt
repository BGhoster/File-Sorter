File Organizer Script
This is a Python script that organizes files in a directory by moving them to specific folders based on their file extensions.

Requirements
Python 3.x
The os module

Usage
Download the script and save it to a desired location.
Open the script in your Python editor of choice.
Set the parent_directory variable to the path of the directory containing the files you want to organize.
Run the script.
The script will create four folders (Image folder, TXT folder, MP3 folder, and MP4 folder) in the specified parent directory using the os module. It will then move files with extensions .txt, .png, .jpeg, .mp3, and .mp4 to their respective folders.
